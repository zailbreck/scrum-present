Kelebihan framework Svelte

  1. Svelte dapat berperan sebagai compiler.
  2. Dapat berjalan dengan menggunakan typescript
  3. Framework memiliki ukuran yang kecil.
  4. Dibandingkan framework lain, untuk mendapatkan hasil yang sama svelte membutuhkan lebih sedikit kode.
  5. Performa nya lebih cepat dibanding framework lain

Kekurangan framework Svelte

  1. Framework Svelte tidak banyak digunakan sehingga pengembangannya agak lambat
  2. Kurangnya dokumentasi membuat framework ini menjadi agak sulit dikuasai.
